### 📌 브랜치
- branch 이름은 feat/{feature name}-{name} 으로 통일 (ex. feat/button-sy)
<br>

### 📌 커밋 메세지
- `feat` 새로운 기능을 추가한 경우
- `fix` 버그를 고친 경우
- `docs` 문서를 수정한 경우
- `refactor` 코드 리팩토링에 대한 커밋
- `rename` 파일 혹은 폴더명을 수정한 경우
- `remove` 파일을 삭제한 경우
- ex) feat: button 추가
